
# E-commerce Project

This project interacts with the FakeStoreAPI, making requests to different endpoints to fetch product data. It provides a user-friendly interface for browsing, filtering, and purchasing products. It offers a responsive design that adapts to different screen sizes.

## Tech Stack

- React.JS
- Material-UI
- Axios
- [FakeStoreAPI](https://fakestoreapi.com/docs): A mock API used to fetch product data
- React Router DOM

## Features

- State management through Context API.
- CurrencyFormatter ensures proper formatting of prices.
- A shopping basket allowing users to decrease or increase the quantity of a particular item.
